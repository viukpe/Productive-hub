# Productive-hub

### Visit the link to view demo --> https://youtu.be/tFdlFZ5XIAQ

## About
Productivity-hub is an improvement on a basic to-do list app by adding an page to show daily task completion, average task weight, and streak progression over time via interactive line and bar plots. 

## How to run locally
- Create virtual environment and install django, plotly, django-crispy-forms and crispy-bootstrap4 using pip or conda
- run `python manage.py runserver` in virtual environment
- View app on --> http://127.0.0.1:8000
